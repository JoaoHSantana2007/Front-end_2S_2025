Títulos e Subtítulos

Dentro do HTML possuímos 6 níveis de títulos. Elas definem os níveis de importância do título no HTML. Por padrão, os navegadores as exibem em tamanho decrescente. Ele possui uma semântica, por padrão devemos usá-las como em um índice de um livro: a tag <h1> deve ser usada apenas uma vez para representar o título da página; as restantes tags podemos usar em uma ordem sequencial lógica.

Parágrafos e Quebra de Linhas

Dentro do HTML, possuímos uma tag feita para escrevermos um bloco de texto: essa tag é a <p>. A cada separação de ideias ou frases relacionadas, o conteúdo deve ser colocado dentro da tag <p>.Porém, o navegador não faz quebras de linha baseadas em como quebramos a linha no código HTML. Por ser um elemento de bloco, o parágrafo ocupa 100% da largura disponível da página. Além disso, o navegador adiciona automaticamente uma margem em cima e embaixo para diferenciar um parágrafo do outro, garantindo a separação visual do conteúdo.

Comentários

Dentro do HTML, possuímos os comentários no código. Eles servem para que os outros desenvolvedores possam saber o que você fez no código e orientá-los nos próximos passos. Além disso, são usados para desativar temporariamente partes do código sem precisar apagá-las.
