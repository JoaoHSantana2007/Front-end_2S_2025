Formatação de Texto

Importância Semântica: <b> e <strong>

A tag <strong> é mais recomendada semanticamente quando queremos dar importância a alguma palavra ou trecho de uma frase.
Ex: Marque a alternativa que está INCORRETA.

Já a tag <b> não é recomendada semanticamente. O HTML recomenda que a usemos somente quando não houver mais nenhuma outra opção.
Ex: Marque a alternativa que está INCORRETA.

Ênfase Semântica: <em> vs. <i>

A tag <em> é usada no HTML para simbolizar que aquela palavra ou trecho de texto possui uma ênfase tônica diferente dos demais, um jeito diferente de falar.
Ex: Posso faltar à aula hoje? Não, você vai para a escola hoje

Já a tag <i> serve para colocar uma palavra ou parte do texto em itálico, usada muito em títulos de obras, palavras estrangeiras e expressões latinas como et al. Ex: Hoje estudaremos o livro Diário de Anne Frank

Tag <mark>

A tag <mark> no HTML serve para marcar uma palavra ou trecho de texto como se passássemos um marca-texto por cima em um livro, sendo usada para realçar uma palavra com importância maior que as demais.
Ex: Este trecho é muito importante para o texto e deve ser marcado

Tag <small>

A tag <small> no HTML serve para que deixemos uma palavra ou trecho de texto menor que os demais da página.
Ex: Essa palavra é menor que as outras.

Tags <del> e <ins>

A tag <del> no HTML serve para dizer que algo foi removido ou trocado no texto.
A tag <ins> no HTML serve para dizer que algo foi inserido no texto.
Ex: A calça custava R$100,00 porém agora ela está custando R$80,00.

Tag <sub>

A tag <sub> no HTML serve para criarmos um texto subscrito no trecho, usada muito em fórmulas matemáticas, químicas e físicas, de forma correta, sem uma incorreta interpretação do leitor.
Ex: H + O2 = H2O

Tag <sup>

A tag <sup> no HTML serve para criarmos um texto sobrescrito no trecho, usada muito em fórmulas matemáticas, químicas e físicas, de forma correta, sem uma incorreta interpretação do leitor, basicamente o contrário da tag <sub>.
Ex: 22 = 4

Bloco de Texto Pré-formatado

                A tag <pre> no HTML serve para que tudo que você digite no código HTML seja replicado no site, diferente da tag <p>.
                 Basicamente, quebras de linhas, espaços, identações, etc. escritas no código serão replicadas na página do site.
